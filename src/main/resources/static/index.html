🖊️ Stationery Management API

RESTful API built with Java and Spring Boot for managing office stationery inventory.

Swagger UI Manage Categories Manage Items

Features

Category Management

Create, update, delete and view stationery categories.

Inventory Management

Manage stationery items including brand, price, quantity and category.

REST API

Supports CRUD operations, pagination, sorting and filtering using Spring Boot.

Technology Stack

Java 17 Spring Boot 3.3.1 Spring Data JPA Hibernate H2 Database Swagger / OpenAPI Maven

Available APIs

GET /api/items
POST /api/items
PUT /api/items/{id}
DELETE /api/items/{id}

GET /api/categories
POST /api/categories
PUT /api/categories/{id}
DELETE /api/categories/{id}

Stationery Management API

Developed using Spring Boot, Spring Data JPA and H2 Database.
